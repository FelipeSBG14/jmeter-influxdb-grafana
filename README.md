# jmeter-influxdb-grafana
Example setup for visualizing JMeter metrics in Grafana using InfluxDB as storage.

# Deploy

```bash
docker compose up -d
```

# Grafana

Default usename/password: admin/admin.

# InfluxDB

Visit:

```
localhost:8086
```

Username/password: admin/admin123.

Initial bucket name: jmeter.
